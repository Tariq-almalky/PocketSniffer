# PocketSniffer

A lightweight packet sniffer written in Python that captures and displays network traffic to help understand how data moves across a network.

---

## Features

- Capture network packets in real time
- Display basic packet information
- Simple and lightweight Python implementation
- Useful for learning networking and cybersecurity fundamentals

---

## Installation

Clone the repository:

git clone https://github.com/Tariq-almalky/PocketSniffer.git

cd PocketSniffer

Run the script:

python PocketSniffer.py

---

## Usage

Run the program from the terminal:

python PocketSniffer.py

The program will start capturing network packets and displaying their information.

---

## Technologies Used

- Python
- Socket Programming
- Networking fundamentals
